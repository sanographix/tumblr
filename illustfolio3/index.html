Illustfolio 3

Free responsive tumblr theme for illustrators

The easiest way to make your own portfolio.

イラストポートフォリオサイトを一瞬で作ろう

Illustfolio 3は、イラストポートフォリオサイトが一瞬で作れる無料Tumblrテーマです。
バージョン3.0は、今までで最も大きなアップデートです。より簡単・より快適に使えるようにしました。

Install Illustfolio 3

デザインを一新

Illustfolio 2で好評だった「イラストポートフォリオサイトが一瞬で作れる」コンセプトはそのままに、テーマのすべての要素を大胆に見直して、完全に新しいデザインに作りなおしました。

プロフィールを表示

自分のプロフィールは、自分の作品と同じくらい大切です。Illustfolio 3ではヘッダーのすぐ下に自己紹介欄を設けたので、経歴や趣味などをより多くの人にアピールできます。

タブレットやスマートフォンでも

ブラウザ幅に合わせて最適化された見た目になります。PCだけでなく、タブレットやスマートフォンでの閲覧にも対応します。

Features

Built for illustrators.

イラストレーターのために設計

カバー画像を着せ替えよう

トップページには、画面いっぱいにカバー画像を表示できます。ヘッダの色も白と黒から選べます。

無限にスクロール

ページ下部までスクロールすると、次のページを自動で読み込むので、イラストが次々と表示されます。

タグの一覧表示

ヘッダから、今までの投稿につけられたタグを一覧できます。特定のタグに絞って見たいときに便利です。

色のカスタマイズ

Illustfolioが選ばれる理由

Illustfolioシリーズは、2012年の誕生以来「イラストレーターの定番テンプレート」として、プロ・アマチュア問わず35,000以上のサイトに採用されてきました。イラストレーターに寄り添い続けて進化を重ねたIllustfolioを、ぜひ体験してください。
